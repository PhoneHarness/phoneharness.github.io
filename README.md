# PhoneHarness Project Page

Static GitHub Pages site for PhoneHarness and PhoneHarness Bench.

## Local Preview

```bash
python3 -m http.server 8080
```

Then open `http://localhost:8080`.

## Structure

- `index.html`: single-page project homepage.
- `assets/paper.pdf`: current paper PDF snapshot.
- `assets/figures/`: project figures exported for the website.
- `assets/traces/`: dynamic trajectory media exported for the website.

## Deployment

This directory is intended to be pushed as a standalone GitHub Pages repository, for example `phoneharness.github.io`.
